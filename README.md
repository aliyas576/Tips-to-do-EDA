🚀 Exploratory Data Analysis (EDA) Complete Guide — Python
A comprehensive, beginner-to-intermediate Jupyter notebook demonstrating Exploratory Data Analysis using Python, built on the classic Titanic dataset from Seaborn. The guide follows the Four Pillars of EDA with well-commented code, interactive visualizations, and a touch of desi Pakistani flair to keep learning engaging.

🎓 Key Insights from Titanic Data
Gender: Female survival rate (~74%) was nearly 4× higher than male (~19%)
Class: 1st class passengers had ~63% survival vs. 3rd class ~24%
Age: Children were prioritized — protective factor for the young
Fare: Higher fare correlated with higher survival (+0.26)
